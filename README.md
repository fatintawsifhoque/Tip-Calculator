# 🧮 Tip Calculator

A clean and intuitive bill-splitting calculator built with **Vue 3** and **Tailwind CSS**. Perfect for splitting restaurant bills, group expenses, or calculating tips instantly.

## 💡 Why I Built This

Splitting bills among friends or colleagues can be confusing—especially when factoring in tips. I created this tool to make the process **fast, transparent, and error-free**. It's a practical utility that focuses on clean UX and real-time calculations.

## ✨ Features

- **⚡ Real-Time Calculations:** Instantly updates tip amounts and per-person totals as you type.
- **🎯 Flexible Tip Options:** Choose from preset percentages (5%, 10%, 15%) or enter a custom value.
- **👥 Split Among People:** Divide the total bill evenly among any number of people.
- **🎨 Beautiful UI:** Modern design with custom brand colors and smooth transitions.
- **📱 Fully Responsive:** Works seamlessly on mobile, tablet, and desktop.
- **🔄 Easy Reset:** Clear all inputs with a single click.

## 🛠️ Tech Stack

- **Framework:** Vue 3 (Composition API)
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Language:** JavaScript
